   - 掉落怪物: 僅追蹤怪物 (trackerA, trackerB)

- 如果圖片檔案不存在，遊戲會使用預設的彩色方塊顯示
- 圖片格式支援 PNG、JPG、GIF 等常見格式
- 建議使用透明背景的 PNG 格式以獲得最佳效果
- 道具會在遊戲中自動浮動動畫，並帶有發光效果
